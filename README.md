# CleanSlate - Clipboard Cleaner

A lightweight & efficient clipboard cleaning tool for GNU/Linux. It helps you maintain privacy & security by automatically clearing the clipboard contents at regular intervals, ensuring that sensitive data doesn't persist longer than necessary.

## Installation
**1. Clone this repo:**
```
git clone https://github.com/StellarSand/CleanSlate.git
```

**2. Move into the project directory:**
```
cd CleanSlate
```

**3. Give executable permissions to the install script:**
```
chmod +x install
```

**4. Run the install script:**
```
./install
```



## Uninstall
If CleanSlate has been installed, you can remove it by:

**1. Clone this repo (if not done already):**
```
git clone https://github.com/StellarSand/CleanSlate.git
```

**2. Move into the project directory:**
```
cd CleanSlate
```

**3. Give executable permissions to the uninstall script:**
```
chmod +x uninstall
```

**4. Run the uninstall script:**
```
./uninstall
```
